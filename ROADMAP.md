- Complete skeleton of test suite for the module
- Further enrich the command line

## [1.0] - 2026-06

- Finalize the API
- Document each submodule extensively

## [1.x]

- Enhance the test suite
- Incorporate user feature requests

- Comprehensive bugfixes
